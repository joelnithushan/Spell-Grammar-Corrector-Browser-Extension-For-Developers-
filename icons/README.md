# Icons

To generate the required icon files:

1. Open `create_icons.html` in a browser
2. Right-click each canvas and "Save image as..."
3. Save as:
   - `icon16.png` (16x16)
   - `icon48.png` (48x48)
   - `icon128.png` (128x128)

Alternatively, you can use any image editor to create icons with:
- Gradient background: #667eea to #764ba2
- White checkmark symbol
- Rounded corners
