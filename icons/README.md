# Icons

The icon files are already generated and included in this directory.

## Regenerating Icons

If you need to regenerate the icons, you have two options:

### Option 1: PowerShell Script (Recommended)
Run the PowerShell script:
```powershell
powershell -ExecutionPolicy Bypass -File generate-icons.ps1
```

### Option 2: HTML Generator
1. Open `create_icons.html` in a browser
2. Right-click each canvas and "Save image as..."
3. Save as:
   - `icon16.png` (16x16)
   - `icon48.png` (48x48)
   - `icon128.png` (128x128)

### Option 3: Image Editor
Use any image editor to create icons with:
- Gradient background: #667eea to #764ba2
- White checkmark symbol
- Rounded corners
- Sizes: 16x16, 48x48, 128x128 pixels
