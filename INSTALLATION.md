# Installation Guide

## Quick Start

1. **Get API Key**
   - Visit [OpenRouter](https://openrouter.ai/keys)
   - Sign up or log in
   - Create a new API key
   - Add credits to your account

2. **Load Extension**

   ### Chrome/Edge
   - Open `chrome://extensions/` or `edge://extensions/`
   - Enable "Developer mode" (toggle in top right)
   - Click "Load unpacked"
   - Select this extension folder

   ### Firefox
   - Open `about:debugging`
   - Click "This Firefox" tab
   - Click "Load Temporary Add-on"
   - Select `manifest.json` from this folder

3. **Configure**
   - Click the extension icon
   - Click "Settings"
   - Paste your API key
   - Click "Test API Key" to verify
   - Click "Save Settings"

4. **Use**
   - Navigate to any webpage
   - Click extension icon
   - Toggle Spell/Grammar check
   - Click "Check Page"
   - Click highlighted errors to see suggestions
   - Click a suggestion to replace

## Generate Icons

Before loading the extension, generate icon files:

1. Open `create_icons.html` in a browser
2. Right-click each canvas and save as:
   - `icons/icon16.png`
   - `icons/icon48.png`
   - `icons/icon128.png`

Or use any image editor to create 16x16, 48x48, and 128x128 PNG icons with a checkmark design.
